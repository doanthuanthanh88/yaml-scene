# yaml-scene
Create scenes to do everything base on yaml file

# Installation
```sh
  docker pull doanthuanthanh88/yaml-scene:latest
```

# Running
```sh
  docker run -it -v /home/testcases:/test doanthuanthanh88/yaml-scene yas /test/YOUR_SCENE.yaml
```
